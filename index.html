Tutorial: How to Choose the Correct Network

Step 1: Identify the dominant location

  • Open the Census file and focus on the ZIP column.
  • Count which ZIP shows up most frequently.
  • Search that ZIP or its city above.
Example: If ZIP 64050 appears eight times and 64151 twice, search 64050 (Lee's Summit, MO within Kansas City MSA).

Step 2: Count enrolled lives

  • Only count EE, ES, EC, EF rows (ignore Waivers).
  • This number defines which MGUs you submit.
Heads-up: Groups under 5 enrolled cannot be quoted on Arlo in Q4 2025.

Step 3: Use the search or wizard

  • Use the quick search for cities/ZIPs/states.
  • Use the guided wizard if you need extra confirmation.
  • Confirm cases on the map.
Need help? Keep the Level Health guides nearby for edge cases.
